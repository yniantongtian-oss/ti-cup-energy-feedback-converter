# Firmware | 固件

This directory contains the STM32 firmware for the energy feedback converter.

## Current Status

- Project skeleton created
- Basic structure defined
- Core control algorithms under development

## Planned Contents

- `Core/` — Main control logic
- `Drivers/` — Peripheral drivers
- `User/` — Application layer (control algorithms, state machine)
- `Inc/` and `Src/` — Standard STM32 project structure

## Build Instructions

Open with Keil MDK or STM32CubeIDE.

> More code will be added progressively during competition preparation.
