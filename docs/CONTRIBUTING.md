# Contributing Guide | 贡献指南

Thank you for your interest in contributing to **ti-cup-energy-feedback-converter**!

感谢你对本项目的兴趣！

## How to Contribute | 如何贡献

### Reporting Issues | 提交 Issue

- Use the GitHub Issues tab
- Clearly describe the problem or suggestion
- Include reproduction steps, expected vs actual behavior
- For hardware-related issues, mention your development board and setup

### Code Contributions | 代码贡献

1. Fork the repository
2. Create a new branch (`git checkout -b feature/your-feature-name`)
3. Make your changes
4. Test thoroughly on hardware if possible
5. Submit a Pull Request

### Documentation | 文档贡献

Improving documentation, tutorials, and examples is highly valued. Even small improvements help other students a lot.

## Code Style | 代码风格

- Follow existing code structure in `firmware/`
- Use meaningful variable and function names
- Add comments for complex control logic
- Keep functions focused and reasonably sized

## Commit Messages | Commit 消息

Use clear and descriptive commit messages, preferably in English or bilingual.

## Questions? | 有问题？

Feel free to open an Issue with the label `question`.
