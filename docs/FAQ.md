# FAQ | 常见问题

## 这个项目是给谁用的？

主要面向正在准备 TI Cup 和相关电子设计竞赛的大学生。也适合想学习功率电子和嵌入式系统的同学。

## 我可以直接用这个代码参赛吗？

目前还在开发中。建议作为参考和学习资源使用，最好结合自己的硬件进行修改和测试。

## 我想贡献代码，但担心写得不好？

欢迎提交！即使是小修改、文档改进或错误修复都很有价值。我会尽快回复并一起完善。

## 这个项目会一直维护吗？

是的。我会在本科期间持续更新，并希望建立小型社区。

## 我能不能用在其他竞赛中？

当然可以。核心的双向功率控制、保护机制、采样算法等内容具有通用性。

## 如何联系作者？

通过 GitHub Issues 提交问题即可。
